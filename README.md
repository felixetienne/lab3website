LabX Framework
============

By Felix-Etienne Tetrault


Informations
============

3-tier starter application.

Include:
- Views (Front-End)
- Services (Back-End)
- Repositories (DAL)


Command line
============

Go to application folder:
$ cd webapps/labx

Install application:
$ npm install

Start application:
$ node app.js
OR
$ forman start

Start grunt sass watch:
$ gunt

Deploy to Heroku:
$ git push heroku master
