LabX Framework
============

By Felix-Etienne Tetrault


Informations
============

3-tier application framework.

Include:
- Views (Front-End)
- Services (Back-End)
- Repositories (DAL)


Command line
============

Go to application folder:
$ cd [your app folder]

Install application:
$ npm install

Start application:
$ node app.js
OR
$ foreman start

Start grunt sass watch:
$ grunt

Deploy to Heroku:
$ git push [heroku] [branch]
